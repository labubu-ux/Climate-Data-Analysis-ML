# 🌍 Climate Data Analysis & Machine Learning

## 📌 Overview
This project analyzes climate data to identify patterns and build predictive models using machine learning.

## 🛠️ Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 🔍 Workflow
1. Data Cleaning (handling missing values)
2. Exploratory Data Analysis (EDA)
3. Feature Selection
4. Model Building:
   - Logistic Regression
   - Random Forest
5. Model Evaluation:
   - Accuracy Score
   - Confusion Matrix

## 📈 Results
- Achieved high accuracy in prediction
- Identified key features affecting climate trends

## 📂 Files
- climate_analysis.ipynb
- dataset.csv

## 🚀 How to Run
Open notebook in Google Colab and run all cells
